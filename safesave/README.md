# safesave
SafeSave implements `io.Writer` and `io.Closer` to atomically overwrite a file, or create it if it did not exist.

## Installation
```sh
go get -u github.com/amy911/env911/safesave
```

## Imports
```go
import "github.com/amy911/env911/safesave"
```
